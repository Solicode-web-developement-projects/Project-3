# Showcase Design - Website Redesign

## Description
This project consists of reproducing a provided mockup in Figma and converting it into a fully functional HTML and CSS web page for Showcase Design. The goal is to enhance the structure and styling while ensuring responsiveness and usability.

## Technologies Used
- **Frontend**: HTML5, CSS3 (Flexbox)
- **Icons**: Font Awesome
- **Design Tool**: Figma

## Features
- Uses semantic HTML5 elements (header, main, footer, nav, etc.).
- Layout implemented with Flexbox for efficient alignment.
- Sticky navigation menu that stays fixed on the top right while scrolling.
- Hover effects applied to navigation links for better user interaction.
- Social media links added to the footer (not included in the original design).
- Responsive and adaptable layout for different screen sizes.

## Installation
### Clone the repository:
```sh
git clone https://github.com/your-profile/showcase-design.git
```

2. Ensure the CSS file is linked correctly.
3. Verify responsiveness and hover effects.
